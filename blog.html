What is the Document Object Model (DOM)?

The DOM represents the whole document as a tree like structure. Every node in the DOM represents an HTML element. It is used for interactive features, manipulate content, validates user input, styling the website dynamically.

How do you select an element from the DOM?

There are several methods, but these 5 used the most:

  • getElementById(id)
  • getElementsByClassName(className)
  • getElementsByTagName(tagName)
  • querySelector(selector)
  • querySelectorAll(selector)

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique for managing events effectively. we can add an event listener to a parent element instead of a every single similar element. It reduces repetition of code, easier to manage, improve performance especially with large content.

How do you manipulate an element's attributes and styles using the DOM?

By DOM, we can access, add, remove, modify attributes and styles.

Manipulate attributes:

  1. getAttribute(attributeName)
  2. setAttribute(attributeName, value)
  3. removeAttribute(attributeName)

Manipulate styles:

  1. style.color
  2. style.fontSize
  3. style.backgroundColor
  4. classList.add(className)
  5. classList.remove(className)
